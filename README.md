<p align="center"><strong>Napet</strong> is an application for managing your pets.
Also for the detection of pests and diseases in rice cultivation with all the advice, care necessary to develop large and small crops.
</p>

## Description

[Nest](https://github.com/nestjs/nest) this application is built with NestJS and PostgreSQL. It's a work in progress for the Quantum-Code team.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
